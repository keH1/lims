13. Какие бывают ЛИМС

 
Классифицировать ЛИМС можно по множеству признаков. Но мы возьмем основные два.
 
1. По сферам применения все ЛИМС можно условно разделить на:
 
  • Производственные – это те, которые используются в собственных лабораториях заводов или различных производств.
  • Для коммерческих лабораторий – эти ЛИМС создавались специально для тех лабораторий, где присутствует внешний заказчик и есть отдел продаж. Данные ЛИМС обычно интегрированы с системой CRM.
  • Для медицинских лабораторий – данный тип ЛИМС разрабатывался специально для обеспечения деятельности медицинских лабораторий, которые работают с биоматериалом и выдают результаты медицинских анализов.
 
2. По техническому подходу выделяют
 
  • Коробочные решения – ЛИМС устанавливается непосредственно на сервер заказчика внутри предприятия. Для этого требуется наличие или покупка оборудования.
  • Облачные решения – ЛИМС располагается на удаленном сервере, а доступ к ней осуществляется на правах аренды. При этом клиенты не покупают оборудование и им не требуется штат сотрудников для его обслуживания.
