КК Шухарта

Контроль стабильности результатов измерений с использованием контрольных карт Шухарта представляет собой графический способ контроля показателей точности результатов измерений для принятия оперативных мер по управлению процессом измерений.

Контрольные карты Шухарта строят для контроля показателей повторяемости, внутрилабораторной прицинзионности, и точности (погрешности) результатов измерений.

Для построения КК Шухарта используют результаты измерений определяемой характеристики в СО, АС, контрольной пробе, используемых для контроля стабильности градуировочных характеристик (для верхнего и нижнего пределов диапазона измерений) или оперативного контроля процедуры выполнения измерений.

  • Образцы контроля - журнал содержит информацию обо всех образцах контроля, созданных в системе ЛИМС U-Lab за все время работы. Из него можно перейти в карточку образцов, используемых для контроля стабильности градуировочных характеристик.
  • Журнал методик и образцов контроля с метрологическими характеристиками - содержит информацию обо всех методиках и назначеных образцах контроля с метрологическими характеристиками. Данный журнал служит для выбора образцов контроля и метрологических характеристик по которым будем контролировать методику. Из него можно перейти в карточку результаты измерений определяемой характеристики.
