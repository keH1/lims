Сервер на Arch Linux

  1. Скачайте установочный образ Arch с https://archlinux.org/download/.
  2. Установите его на флеш-носитель при помощи Rufus.
  3. Зайдите в BIOS/UEFI и отключите Secure boot и Fast/Quick boot.
  4. Выберите загрузку с флеш-носителя (через boot menu).

  1. Осуществите загрузку с флеш-носителя.

  1. Введите команду # archinstall

  3. Выберите жесткий диск

  1. Добавьте дополнительные пакеты, введя: git nano mc wget htop openssh man-db apache mariadb zip unzip libreoffice-still libreoffice-still-ru jre-openjdk jdk-openjdk

  1. Установите конфигурацию сети: "скопировать данные из установочного образа".

  1. Выберите местную временную зону.

  1. Нажмите Установить (install)

  1. После установки на вопрос "Открыть окружение chroot?" ответьте "да".

  1. Введите команду # nano /etc/vconsole.conf

    Откроется файл vconsole.conf
    Измените его содержимое на:
    FONT=cyr-sun16
    KEYMAP=ru
    Сохраните изменения, нажав Ctrl+S
    Выйдите из редактора, нажав Ctrl+X

  1. Введите команду # nano /etc/pacman.conf

    Откроется файл pacman.conf
    Включите поиск, нажав Ctrl+W
    Введите "ParallelDownloads" без кавычек
    Нажмите Enter
    Удалите символ "#" в начале строки
    Сохраните изменения, нажав Ctrl+S
    Выйдите из редактора, нажав Ctrl+X

  2. Последовательно введите следующие команды:

    # mkdir /home/user/public_html
# mariadb-install-db --user=mysql --basedir=/usr --datadir=/var/lib/mysql
# systemctl enable httpd.service
# systemctl enable sshd.service
# systemctl enable mariadb.service
# exit
# reboot
